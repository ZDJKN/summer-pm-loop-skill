# summer-pm-loop-skill

> **一句话：一个像老练项目负责人（PM）一样，陪你跑客户项目的 AI 助手。**
> 接需求、写周报、算账、客户闹脾气、项目要复盘——你把事丢给它，它用一套从真实项目里磨出来的打法，给你**能直接照着做**的话术、判断和下一步。

作者：**ZDJKN · Summer · Mr Wu**

---

## 是什么

不是又一个「项目管理软件」，也不是讲大道理的管理课。

它把**一线服务型 PM / 主R 的实战打法**——怎么接需求不挖坑、客户不回怎么办、这单接不接、毛利怎么算、客户发飙怎么救火、周报怎么写老板才看得懂——**装进了一个 AI 助手**。你说人话，它给你能落地的动作。

## 有啥用（专治这些头疼事）

| 你遇到的事 | 它给你 |
|---|---|
| 客户甩来个模糊需求，怕接了挖坑 | 帮你拆成「做什么 / 不做什么 / 什么算完成」，还给你一句话发回客户确认 |
| 客户好几天不回，干等着急 | 一条不掉价又推进的催法（给选择题 + 你先动起来） |
| 这单到底接不接、能赚钱吗 | 直接帮你算毛利，低于红线就提醒别接 |
| 客户突然发飙 / 不满意 | 12 步救火：先接情绪再解决，话术给到位 |
| 写周报像流水账，老板看不懂 | 教你「结论先行」，一句话说清进度 + 要老板做什么 |
| 项目一堆，记不清哪个要爆雷 | 一盘点，从客户/团队/成本/进度多个角度告诉你「最该担心哪个、谁去动」 |

## 怎么用

装好后**直接说人话**，不用记命令。比如：

- 「我接了个需求，帮我拆一下」
- 「客户不回消息，怎么推进」
- 「这单 30 万、4 个人做，能接吗」
- 「客户不满意，帮我想想怎么回」
- 「帮我写这周的周报」
- 「盘一下这个项目有没有风险」

## 哪里用 / 什么时候用

- **哪里**：任何支持 AI skill 的工具里（Claude Code、Codex 等）。
- **什么时候最值得用**：
  - 🟢 **接新需求 / 新项目的第一刻** —— 别凭感觉答应，先让它帮你拆透。
  - 🔥 **客户那边出状况**（不回 / 改需求 / 不满意）—— 它最擅长这个。
  - 📤 **要对外发东西之前**（周报、给客户的话、报风险）—— 帮你说对。
  - 💰 **算钱的时候**（接不接、报多少）。
  - 📅 **每周盘一次**项目健康度，提前发现要爆的雷。

## 效果如何

- 把「凭感觉、临场慌」变成「有标准动作、有话术、有数」。
- 实战压测过：有人用它跑了一整天客户项目（救火、催反馈、定方案、写同步），全程在线。
- 它的判断不是空话，是从真实项目里一个坑一个坑踩出来的——比如「补人不能说成换人」「频率没写明不能算毛利」「客户不回要给选择题」。这些都是项目里会**真救命**的小动作。

## 上手（3 步）

1. 把这个文件夹放进你 agent 的 skills 目录（Claude Code：`~/.claude/skills/`；Codex：`~/.codex/skills/`）。
2. 开个新对话，说一句上面的场景，它就接管。
3. **想先看个例子？** 打开 `projects/_DEMO/` —— 一个填好的虚构项目（星瀚气泡水），你一看就懂它能产出什么。

> 想让它**自动**抓飞书/钉钉群消息、定时盘点？见 `INSTALL.md`（可选，不影响基本使用；默认手动就够用）。
